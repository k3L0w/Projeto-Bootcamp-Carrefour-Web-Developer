Web-server = servidor de dados

2 tipos: estático (servidor físico, geralmente alocados em locais apropriados)/ dinâmico (softwares no servidor físico, são do tipo: file server, application server, database...)

Difícil disponibilizar apenas um tipo de servidor. 

Um Site/aplicativo, dados ou banco de dados precisam necessariamente estar hospedado em um servidor para ser acessado.

Exemplo de um Web-service
Um iterface para fazer requisições e consulta em bancos de dados inacessíveis
